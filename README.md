## 📂 Repository Contents & Structure

The repository is modularly structured into lightweight, browser-optimized architecture components:

* **`index.html`** – Main landing application interface containing the Hero banner, custom About Me summary, and technical skills grid.
* **`Experience.html`** – Dedicated standalone engineering timeline highlighting software development milestones, Data Structures and Algorithms (DSA) progression, and cloud training.
* **`projects.html`** – Architectural project showcase featuring development challenges, engineering solutions, and structural links.
* **`style.css`** – Centralized stylesheet handling responsive grids, layout transitions, typography variables, and modern dark-theme aesthetics.
* **`script.js`** – Modern JavaScript engine controlling interactive webpage behaviors, DOM manipulations, and interface mechanics.
* **`Images/`** – Asset management directory containing optimized interface graphics and professional profile media.
* **`Resume.pdf`** – Up-to-date professional background resume configured for automatic download optimization.
